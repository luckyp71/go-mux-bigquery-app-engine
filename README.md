# Go, Gorilla Mux, Big Query, and App Engine
A simple example on how to integrate big query and golang as well as deploy the app to google's app Engine

## Prerequisites:
1. Go 1.11.2
2. Gorilla Mux
3. App Engine
4. Big Query
5. SonarQube
